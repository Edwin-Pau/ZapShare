About ZapShare

Our Story and Mission

What is ZapShare all about?

  1. We believe we can change the world for the better by increasing the adoption of electric vehicles in our cities.
  2. We understand the pain points of owning an electric vehicle; we are building techonology that improves the user experience.
  3. We believe in a peer-to-peer network system, where our users can engage directly with other electric vehicle owners.
Environmentally Friendly Transport
Electric vehicles produce zero emissions, making them an ideal choice for mid-to-long range trips around the city.
Clean and Renewable Energy
Electric vehicles go hand-in-hand with renewable energy sources. With more EV's on the roads, we can be less reliant on fossil fuels.
Clean Air
By increasing the adoption of EV vehicles, we are able to lower the number of gas powered cars in our cities that pollute our air.
Healthy and Equitable Economy
Anyone with an available charger can participate in our network as a host and receive peer-to-peer payments for their services provided.

Our Road Map

Completed Features and Future Development

We have completed phase 1 of the project on April 9, 2019. Features incorporated into this release are marked at 100% completion. Additional features will be added as we continue development of the web app. Below is a list of features and developments that we are currently working on. Please reach out to the ZapShare team if you have any suggestions or feedback.

  1. A user and host login system to access and retrieve stored data using Google Firebase.
    100%

  2. A Map page using Google Maps API, with dynamic markers that show host locations nearby.
    100%

  3. A dashboard for users to access various features and options available to them.
    100%

  4. Unique user profile section that allows users to create and store profile information.
    100%

  5. A booking and reservation system that allows users to reserve a charger on the network.
    10%

  6. A host location review system, allowing users to rate hosts based on experience.
    20%

  7. A guide section added to the website, giving users more information about electric vehicles.
    30%

  8. A peer-to-peer payment system, transacted securely on the ZapShare platform.
    10%

  9. A notification and chat system, allowing users and hosts to communicate to each other.
    5%
